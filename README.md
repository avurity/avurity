# üí´ About Me:
I work on machine learning and computer vision, mostly on problems that show up in real systems. A lot of my work is around image and video data from smartphones, where things are noisy, inconsistent, and hard to control. I build models that can handle this kind of data and still behave reliably. I spend most of my time building end-to-end pipelines: collecting and cleaning data, training models, checking where they fail, and making them fast enough to run in practice. I have worked on image classification, detection, and segmentation, especially for security and liveness problems. I also work on ways to update trained models when data needs to be removed or corrected.

Outside vision, I build ML systems like document processing pipelines, recommender models, and LLM-based tools. I care about clear code, repeatable experiments, and systems that are easy to debug and maintain.
